# habesha-names

Ethiopian/Eritrean name intelligence for Python: fidel script handling,
transliteration, spelling-variant generation, name parsing, and
patronymic-aware fuzzy matching.

**Status: pre-alpha (v0.1 in development).** APIs and linguistic data are not
stable yet.

- Zero runtime dependencies (stdlib only)
- Deterministic and explainable — built for KYC/AML, remittance, HR, and
  entity-resolution pipelines
- Fully typed (`py.typed`)

See [ARCHITECTURE.md](ARCHITECTURE.md) for the design.
